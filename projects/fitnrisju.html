fit.n.risju

Lifestyle & fitness portal

Project summary

This is a website that I built for my sister and her partner who are working on a blog around healthy food and fitness. They wanted the page to be available in two languages and needed a content management system (CMS) so that they could add, edit, and delete content to the site themselves.

Core technologies

React, Gatsby, Netlify, HTML, CSS, Javascript

Main challenge

This was my first project that involved component-based architecture. I also used React and Gatsby for the first time. Using this technology made it possible to integrate a static site generator (Gatsby) and CMS (Netlify CMS) so that my sister and her partner can create content and new sub-pages themselves. I was really happy to have great mentoring for this project, because I had never worked with React or Gatsby before.

Key learnings

I learned a ton about component-based architecture and its advantages in this project. Using Javascript to dynamically choose certain page content based on the language selection of the user was also an interesting aspect to implement.
