EcoRoute

Route and CO2 calculator

Project summary

EcoRoute is a route calculator that gives users insights on the CO2 impact of their individual travel choices. The application is making it easy for users to learn more about the environmental impact of their travel choices by helping them understand the CO2 emission of their trip. They can simply choose what travel options they want to consider for their route, enter their destination and origin, and the app is giving them all the insights they need to make a green choice.

Core technologies

Javascript, HTML, CSS

Main challenge

This is a team project; my core responsibilities were writing the script to toggle all the travel options and calculate the CO2 emission based on the output of the mapping API as well as the styling of the user interface. The main challenge for me was to make the buttons clickable in any order and edit to the results accordingly. In order to make the application as user friendly as possible, I wanted the buttons to be toggles so the users can easily adjust their choices dynamically.

Key learnings

This was my first team project, so I learned a ton about collaborating with others and Git. It was great to see the individual pieces come together and work in a team on this project. I'm excited to continue to work on this application and add more features to it!
